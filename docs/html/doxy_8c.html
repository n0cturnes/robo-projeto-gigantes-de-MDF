Codigo_GiganteDeMdf 1.0
O código controla um robô de MDF, gerenciando a interação com motores e LEDs por meio de interrupções. Ele monitora a entrada do ADC para detectar mudanças de estado, gerenciando vidas e invulnerabilidade. Além disso, controla o movimento e os LEDs com comandos recebidos via USART e ajustes temporais de invulnerabilidade e efeitos visuais.

Interrupção do ADC

Esta rotina é acionada quando a conversão do ADC é concluída. Ela lê o valor do ADC e verifica se ele está acima do limiar (THRESHOLD). Caso afirmativo, o LED correspondente é apagado, o número de vidas é decrementado, e a invulnerabilidade é ativada. O contador é reiniciado e a conversão do ADC é iniciada novamente.

Interrupção do Timer0

Esta rotina é acionada a cada 1 ms pelo Timer0. Ela gerencia o estado de invulnerabilidade e o controle do LED piscante do laser. Durante o tempo de invulnerabilidade, os motores são desligados. Quando o tempo de invulnerabilidade expira, os motores são ligados novamente. Também controla o LED de laser, invertendo seu estado a cada intervalo de piscada.

Interrupção do botão (PB0)

Esta rotina é acionada quando o botão é pressionado (PB0). Ela reinicia o contador de vidas e acende os LEDs para indicar que o sistema foi resetado.

Interrupção de recepção USART

Esta rotina é acionada quando um dado é recebido via USART. Ela verifica o valor recebido e, dependendo do comando recebido, controla os motores e LEDs do sistema:

  • 'F' faz os motores irem para frente.
  • 'B' faz os motores irem para trás.
  • 'S' para os motores.
  • 'L' liga o motor 1 e desliga o motor 2.
  • 'R' liga o motor 2 e desliga o motor 1.

void TIMER0_init ( )

Inicializa o Timer0 para gerar interrupções a cada 1 ms.

O Timer0 é configurado em modo CTC (Clear Timer on Compare Match) com prescaler 64, gerando uma interrupção a cada 1 milissegundo. Essa interrupção é usada para controlar o estado de invulnerabilidade e o controle do laser piscante.
